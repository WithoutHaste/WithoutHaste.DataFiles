# DataFiles.Markdown Summary

This document contains a list of Markdown constructs that this library can interpret/handle.

* Headers
* Text
  * Plain Text
  * Bold Text
  * Italic Text
  * Bold-Italic Text
* Line breaks
  * Single line breaks
  * Paragraph breaks
* Hyper-links
* Code
  * In-line code
  * Code blocks
  * Language specification
* Tables
* Lists
  * Numbered lists
  * Bullet lists
  * Nested lists
